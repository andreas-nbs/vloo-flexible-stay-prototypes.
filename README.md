# VLOO Flexible Stay Prototypes

This repository contains the standalone Flexible Stay booking calendar prototypes.

## Files

- `host-portal-flex-booking-calendar-prototype.html` - host Space Manager calendar prototype for managing Flexible Stay availability and desk capacity.
- `user-facing-flex-booking-calendar-prototype.html` - user-facing Flexible Stay booking calendar prototype.

## Scope

- Flexible Stay prototypes only.
- No Extended Stay calendar work is included here.
- No backend code.
- No production booking or payment logic.

`vloo-project-01` should be treated as old/prototype-only work and should not be used as the base for new Flexible Stay prototype work.
